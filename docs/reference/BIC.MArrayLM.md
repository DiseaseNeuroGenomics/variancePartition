# BIC from model fit

BIC from model fit

## Usage

``` r
# S3 method for class 'MArrayLM'
BIC(object, vobj, ...)
```

## Arguments

- object:

  result of `lmFit()` or
  [`dream()`](http://DiseaseNeurogenomics.github.io/variancePartition/reference/dream-method.md)

- vobj:

  `EList` used to fit model

- ...:

  See [`?stats::BIC`](https://rdrr.io/r/stats/AIC.html)

# Residuals for result of dream

Residuals for result of dream

## Usage

``` r
residuals.MArrayLM2(object, y, ..., type = c("response", "pearson"))
```

## Arguments

- object:

  See [`?stats::residuals`](https://rdrr.io/r/stats/residuals.html)

- y:

  `EList` object used in
  [`dream()`](http://DiseaseNeurogenomics.github.io/variancePartition/reference/dream-method.md)

- ...:

  See [`?stats::residuals`](https://rdrr.io/r/stats/residuals.html)

- type:

  compute either response or pearson residuals
